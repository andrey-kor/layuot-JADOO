## Возможности:
- Валидация e-mail;
- Слайдер с отзывами;
- Выпадающее меню;
- Адаптивная вёрстка;
- Анимированные элементы при скролле (при ширине экрана от 600px до 1060px);

- HTML
- CSS;
- JavaScript.
